Zdi

Buňky, přes které nelze vést cestu. Musíš je obejít.

Portály

Dvojice propojených bodů. Když cesta vstoupí do jednoho portálu, vystoupí z druhého — barva zůstává.

Měniče barev

Při průchodu touto dlaždicí se barva cesty změní (např. z červené na modrou). Cílový endpoint musí odpovídat výsledné barvě.
